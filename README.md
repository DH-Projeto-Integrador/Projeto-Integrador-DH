# Start Project

```bash
# clone repository
git clone https://github.com/DH-Projeto-Integrador/Projeto-Integrador-DH.git

# install dependecies
npm install

# run application (terminal primary)
npm run dev

# run tailwindcss (terminal secondary) 
npm run tailwindcss
```
